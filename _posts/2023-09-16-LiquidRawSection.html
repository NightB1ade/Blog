Introduction

When typing up the other post for today, I wanted to include a code section within <code><pre> tags. This code happenend to have {% raw %}{% tags %}{% endraw %} which then the Jekyll static site genertor was attempting to parse as these look like liquid tags. Thus I had to find out how to escape this in liquid syntax.

The code for this in Liquid is: {% raw %}{% raw %}Non-parsed{% {% endraw %}{% raw %}endraw %}{% endraw %}
